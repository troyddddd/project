To train the model, run train.py, which contains the training loop.
Model is defined in qa_model.py, data processing code are in qa_data.py.
evaluate.py and data/squad_preprocessing.py are provided evaluation and preprocessing scripts.
data directory contains the preprocessed data.
embedding directory contains embedding files and ids of words to embeddings.
src directory contains features and needed files ported from tensorflow 1.3.
